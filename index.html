1. Introduction

BudgetFlow ("Company," "we," "us," "our," or "Service") is committed to protecting your privacy and ensuring you have a positive experience on our platform. This Privacy Policy explains how we collect, use, disclose, and otherwise handle your information when you use our mobile application, BudgetFlow, available on Android and iOS (the "App" or "Service").

Please read this Privacy Policy carefully. If you do not agree with our policies and practices, please do not use our Service.

2. Information We Collect

2.1 Information You Provide Directly

  • Account Registration: Email address, password, first and last name, phone number (optional)
  • Financial Information: Transaction details (amount, date, time, category), transaction descriptions, budget limits and goals, income and expense records, recurring transaction information
  • Profile Information: User preferences, notification settings, language and currency preferences, display preferences

2.2 Information Collected Automatically

  • Device Information: Device type, OS version, device identifiers, storage and memory info
  • Usage Analytics: Features accessed, time spent, app crashes and error logs, performance metrics
  • Network Information: IP address, connection type, approximate location (country/region level)

2.3 Information from Third Parties

  • Firebase (Google): Authentication logs, security event data, app analytics, crash reporting
  • RevenueCat (if premium): Purchase history, subscription status

3. How We Use Your Information

  • Core Service: Create accounts, authenticate logins, store financial data, generate reports, calculate budgets
  • Communications: Password reset emails, feature announcements, account confirmations, security alerts
  • Service Improvement: Analyze usage patterns, identify issues, improve performance, develop features
  • Security: Detect fraud, prevent unauthorized access, comply with legal obligations

4. Email Communications

We may send you emails for:

  • Transactional: Account confirmation, password reset, security alerts, account changes
  • Informational: New feature announcements, app updates, performance tips (can opt-out)
  • Marketing: Special offers, product recommendations (opt-in only)

You can manage email preferences by clicking "Unsubscribe" in promotional emails or adjusting settings in your account.

5. Data Storage and Security

Storage Locations:

  • Firebase Authentication (Google Cloud) for login
  • Firebase Firestore (Google Cloud) for transaction data
  • AsyncStorage (local device) for caching

Security Measures:

  • HTTPS/TLS encryption for data in transit
  • Firebase security rules limiting access to your data only
  • Password hashing (not stored in plaintext)
  • Secure token storage
